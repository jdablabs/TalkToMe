# Privacy Policy – Talk to Me 

**Last updated: November 2025**

**Talk to Me ** does **not** collect, store, or share any personal data.

- No user accounts  
- No analytics or tracking  
- No ads  
- No internet permission (except for opening Google Maps when the user chooses)  
- All cards, schedules, settings, and photos are stored **only on the device**  
- Phone calls use Android’s built-in CALL_PHONE permission (only when the user taps “CALL”)  
- Location is used only to open Google Maps to the saved home address (when the user taps “HOME”)

This app was made by a dad for his autistic son, Maddox.  
Nothing ever leaves the phone.
